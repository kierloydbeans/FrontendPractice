Day 5 of HTML Practice

In this 5th day of me relearning HTML, I am aiming to rewrite code from the previous day from scratch. I wanna build it without checking it. I wanna build muscle memory. At the end of this journey, I want to be able to understand frontend with little to no effort.

What have I learned so far?

So far, in the 5 days that I am learning HTML, I have remembered all these concepts:

  • HTML Basics
  • Semantics
  • Rewriting from memory
  • Consistency for everyday

Why am I doing this?

As a third year BSIS student, I feel like I am being left out. I feel like people are advancing way too fast and I feel like I am being left behind. That is why I decided to make a move. To do this daily coding practice. To build muscle memory, to build a habit. To become better. As a student, and as a future asset in the workforce.

In the technical perspective, as I am also studying Python backend programming, I am relearning Frontend so I will be ready when I get into Python Frameworks such as Django, FastAPI, and others. I thought that if i could not comprehend basic frontend code, I might have a hard time understanding frameworks if all i do is backend.

I am also doing this to not rush the process. I believe that great things take time. As they say, "Rome wasn't built in a day." So I am taking my time, ensuring that each day, when I learn new things, I really store it in my brain. So that I can prevent that "I am progressing. But am I really learning?" mentality. Because that thought eats me up so much inside. I think this is a symptom of imposter syndrome, lol.

Daily Rules

In this daily frontend practice, I follow these simple rules:

  1. Code everyday. Even ugly code counts.
  2. Start from scratch. No overwriting existing code.
  3. Little to no googling.
  4. 20 to 30 minutes allotted time.
